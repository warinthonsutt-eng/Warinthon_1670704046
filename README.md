วิธีการติดตั้งและรันโปรแกรม

1: ติดตั้ง NetBeans ดาวน์โหลด Apache NetBeans ได้จากเว็บไซต์ทางการ ติดตั้งโปรแกรมและเลือกการติดตั้งพร้อม JDK หากยังไม่มี เปิด NetBeans หลังติดตั้งเสร็จ ขั้นตอนที่ 2: เปิดโปรเจกต์ ไปที่เมนู File > Open Project เลือกโฟลเดอร์โปรเจกต์ที่ชื่อ Final_Project_CS436_BMICal รอให้ NetBeans โหลดโปรเจกต์จนเสร็จ ขั้นตอนที่ 3: ตั้งค่า Main Class ก่อนรัน คลิกขวาที่ไอคอนรูปแก้วกาแฟด้านข้างของโปรเจกต์ เลือก Properties ไปที่หัวข้อ Run ตั้งค่า Main Class เป็น com.mycompany.test_1.MainFrame กด OK ขั้นตอนที่ 4: รันโปรแกรม คลิกขวาที่โปรเจกต์ เลือก Run ระบบจะเปิดหน้าต่าง Main UI ของ BMI Calculator

วิธีใช้งานแอปพลิเคชัน หน้า Main UI เป็นหน้าหลักที่มีเมนูด้านบน ได้แก่ BMI Form BMI Info BMI History Exit เพื่อปิดโปรแกรม หน้า BMI Form ใช้สำหรับกรอกข้อมูลและคำนวณ BMI ประกอบด้วยช่องกรอก First Name Last Name Weight Height BMI (แสดงผลลัพธ์อัตโนมัติหลังกด Calculate) ปุ่มที่ใช้งาน Calculate: คำนวณค่า BMI จากน้ำหนักและส่วนสูง Save: บันทึกข้อมูลลงไฟล์ BMIData.bin Close: กลับไปหน้า Main UI หน้า BMI Info หน้าสำหรับแสดงข้อมูลและคำอธิบายเกี่ยวกับดัชนีมวลกาย ประกอบด้วยพื้นที่แสดงรูปภาพและคำอธิบายข้อความเกี่ยวกับค่า BMI ในแต่ละช่วง ปุ่ม Close สำหรับกลับไปหน้า Main UI หน้า BMI History หน้าสำหรับดูประวัติ BMI ทั้งหมดในรูปแบบตาราง ข้อมูลประกอบด้วย First Name Last Name Weight Height BMI ปุ่มที่ใช้งาน Delete: ลบข้อมูลที่เลือกจากตารางและจากไฟล์ Close: กลับไปหน้า Main UI

ข้อกำหนดระบบที่ต้องมี Java JDK 17 หรือใหม่กว่า Apache NetBeans (รุ่นใดก็ได้ที่รองรับ JDK 17) ระบบปฏิบัติการ Windows, macOS หรือ Linux# Warinthon_1670704046
